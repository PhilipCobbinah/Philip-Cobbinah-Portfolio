# Philip Cobbinah - Portfolio Website

This is the personal portfolio website for Philip Cobbinah.

## Structure

- `index.html` - Main landing page
- `portfolio-details.html` - Portfolio projects showcase
- `service-details.html` - Services offered
- `starter-page.html` - Template for additional pages
- `assets/` - Static assets directory
  - `css/` - Stylesheets
  - `js/` - JavaScript files
  - `img/` - Images
  - `vendor/` - Third-party libraries
  - `cv/` - CV/Resume files

## Usage

Open `index.html` in a web browser to view the portfolio.

## Notes

- Add your CV as `Philip_Cobbinah_CV.pdf` in the `assets/cv/` folder
- Place images in the `assets/img/` folder
- Add third-party libraries in the `assets/vendor/` folder
